## 추천 음식 리스트, 카카오톡 나에게 메세지 보내기로 메세지 전송

### Used API
- Naver local API
- Kakao API


### Used laungage
<img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=Python&logoColor=white"/> 


### 기능
- 랜덤으로 {음식} 리스트 만들기
- {지역} 이름을 검색하면 {지역} {음식} 맛집으로 네이버에 검색 (예 - 서면 김밥 맛집)
- 결과 3개를 카카오톡 나에게 메시지 전달로 전송
